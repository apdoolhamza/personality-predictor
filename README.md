## What does it do?

It predicts your **MBTI Personality Type** (like INTJ, ENFP, ISTP, etc.) based on what you write.
It also shows an easy-to-understand explanation of:
- Your type
- What it means
- A famous quote ...

## Dataset
This AI was trained on a public dataset made from over 8,000+ people‚Äôs writings, where each person‚Äôs MBTI type is known.
